Blog [1] - The_Inception (02/07/2021)

This is my very first blog. I was writing code the other day for this webpage. One thing that you'll notice to be different from the homepage would be my name at the top and the profile picture, I scaled it down a notch and it now appears on the top left followed by a horizontal line ,which I have to say, looks very good. Everything else is kinda similar. The CSS I wrote for this page will serve as a template. This is convenient because I can just use the same css file for everyother blog I'm going to write. And a single change in the css file will subsequently change how everyother blog will look. I will have to change it manually if I had separate CSS files, but I don't.

I've decided to keep the 'Change Background' and 'Set Dark Mode' buttons intact for this document too. I really like how the lime-cyan colour gradient looks, which is why I'm using that here as well. The horizontal lines in this document, in my opinion, make the page look much cleaner. The displayed text is centered and not extremely close to the left and right margins.
Most of the time spent was on CSS media queries. (gotta make the design look good for the mobile users who come by this webpage). It was a pain, I just wish everyone uses a device with the same resolution.

Another thing you'll notice is that the URL goes like "blog/the_inception.html". I've been thinking about using numbers as filenames; to make it easier to realise which blog you're on. That was when I decided to display the number at the very beginning of this page ('Blog [1]')
I'll probably do the same for every other blog too. There is one term for using the index as the file name. I belive that term would be 'incremental counting systems' where everfile would be named as 1.html, 2.html.. and so on. Giving the filename as the title of the blog would be much cooler. There's also a 'breadcrumb' at the top of this page which is basically the path of the current document from the root folder. ("index.html/blog/the_inception.html" but I used arrow marks between them instead of slashes).

Anyway, I had fun doing this.
Thats all I have to say for now. Thankyou for reading.
